# Telco Customer Churn Dashboard

## Overview:
This project aims to analyse churn data from a company's customer-base data, using Excel and PowerBI.

The objective is to identify key retention drivers and churn reasons.

The findings were used to create a client-ready dashboard and provide actionable business recommendations and insight to reducing churn rates.

## Aim of The Analysis:
To answer questions such as:
- Why are customers leaving the platform?
- Which customer segments are most likely to churn?
- How long do customers typically stay active?
- What actions can improve customer retention?
